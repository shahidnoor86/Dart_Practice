# Dart_Practice

1. Write a program in Dart that finds simple interest. Formula= (p * t * r) / 100
2. Add only numbers from a string
3. Write a program to check if a number exists in a list and print an appropriate message. 
4. Write a function to calculate the factorial of a number.
5. Implement a function that takes a list of strings and returns a new list with the strings reversed.
6. Write a program to print the current time after a 3-second delay using Future.delayed().
7. "Make a two-player Rock-Paper-Scissors game against computer.
Ask for player's input, compare them, print out a message to the winner."
